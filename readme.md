
# My Darknet

This repository is an attachment to this presentation on my YouTube channel.
[Link on YouTube](http://youtu.be/4AIenJGPFp0)

The **My Darknet** project is my own project for creating my own darknet environment using Oracle Application Express technology.

This repository includes all the BASH programming code for the Linux Ubuntu operating system, as well as the Oracle PL/SQL code used in my presentation. I'm also including a ready-to-import APEX application called "My Darknet."
